## 🔍 What is rail-fence-cipher-python?

rail-fence-cipher-python is a simple program made in Python. It lets you try out the Rail Fence cipher, an old way to hide messages. This cipher works by writing a message in a zigzag pattern across a few rows and then reading it row by row.

This program helps you see how the cipher works by letting you both lock (encrypt) and unlock (decrypt) messages. It’s designed to teach you about this classic method of scrambling text without needing to code or use complicated tools.

## 🛠️ How to Use the Program

Once the program is running, you will see a simple menu:

- Type **1** to encrypt a message.
- Type **2** to decrypt a message.
- Type **3** to exit the program.

### Encrypt a message

1. Choose option 1.
2. Enter the message you want to lock.
3. Enter the number of rails (rows) to use for the zigzag. Usually between 2 and 5.
4. The program will show the scrambled message that hides your original text.

### Decrypt a message

1. Choose option 2.
2. Enter the scrambled message.
3. Enter the number of rails used during encryption.
4. The program will show the original message.

### Exit

Choose option 3 to close the program.

## 📚 Additional Information

If you want to learn more about the Rail Fence cipher or how transposition ciphers work, here are some simple points:

- The cipher writes the message letters diagonally across multiple rows.
- Reading rows horizontally creates the encrypted text.
- This method hides the original letter order but does not change the letters themselves.
- It is not secure for serious protection, but great for understanding cipher basics.
